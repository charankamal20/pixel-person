<div class="markdown prose w-full break-words dark:prose-invert dark"><hr><h1><strong>Avatar Generation API</strong></h1><h2><strong>Overview</strong></h2><p>Avatar Generation API is a modern, lightweight service that generates cartoon avatars using deep learning models. It supports various features like configurable image sizes and transparent backgrounds. Designed for seamless integration, this API provides an efficient way to create unique avatars for games, profiles, or digital art.</p><hr><h2><strong>Features</strong></h2><ul><li>Generate high-quality cartoon avatars.</li><li>Supports multiple resolutions: <code>96x96</code> (small) and <code>128x128</code> (large).</li><li>Transparent background support for flexible usage.</li><li>RESTful API powered by FastAPI for easy integration.</li><li>Deployed using <code>ngrok</code> for accessible endpoints.</li></ul><hr><h2><strong>Tech Stack</strong></h2><h3><strong>Backend</strong></h3><ul><li><strong>Language</strong>: Python</li><li><strong>Framework</strong>: FastAPI</li><li><strong>Libraries</strong>:<ul><li>TensorFlow/Keras: For loading and using the deep learning models.</li><li>PIL: For image processing and transparency support.</li><li>NumPy: For handling noise input to the generator.</li></ul></li></ul><h3><strong>Frontend</strong></h3><p>Implemented with Next.Js, created for high performance and creative UI. Deployed on Vercel.</p><hr><h2><strong>Neural Network Model</strong></h2><p>The core of the application is a <strong>Deep Convolutional Generative Adversarial Network (DCGAN)</strong>.</p><h3><strong>Architecture</strong></h3><ul><li><p><strong>Generator</strong>:<br>The generator is a deep convolutional neural network that transforms a random noise vector (latent space) into meaningful images. It uses techniques like upsampling and transposed convolutions to create sharp and realistic images.</p></li><li><p><strong>Discriminator</strong>:<br>Although not used in the API, the discriminator was a part of the training pipeline. It classifies images as real or fake, helping the generator improve iteratively.</p></li></ul><h3><strong>Training Details</strong></h3><ul><li><p><strong>Dataset</strong>:<br>The model was trained on the <strong>Cartoon Set</strong> dataset, a large collection of 2D cartoon avatar images, curated for generating diverse facial features and styles.</p><ul><li>Dataset Link: <a rel="noopener" target="_new" style="--streaming-animation-state: var(--batch-play-state-1); --animation-rate: var(--batch-play-rate-1);"><span style="--animation-count: 2; --streaming-animation-state: var(--batch-play-state-2);">Cartoon</span><span style="--animation-count: 3; --streaming-animation-state: var(--batch-play-state-2);"> Set</span><span style="--animation-count: 4; --streaming-animation-state: var(--batch-play-state-2);"> Dataset</span></a>.</li><li>Augmented to handle various attributes like facial shapes, eye types, and hairstyles.</li></ul></li><li><p><strong>Input Dimensions</strong>:</p><ul><li>Latent vector (<code>SEED_SIZE</code>): 100 for 96x96 images and 128 for 128x128 images.</li><li>Output Dimensions: Either <code>96x96x3</code> or <code>128x128x3</code>.</li></ul></li><li><p><strong>Optimization</strong>:</p><ul><li>Optimizer: Adam with learning rates tuned for GAN training stability.</li><li>Loss
